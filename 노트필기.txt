생성자?
-객체의 인스턴스를 생성할때 강제로 호출되는 메서드, new 연산자 뒤에는 반드시 생성자 
 호출이 있어야 한다(java와 동일)

멤버변수(Property)?
-인스턴스가 보유한 변수를 말한다.
 new에 의해 인스턴스가 생성될 떄 태어나서 해당 인스턴스와 생명을 함께 한다.
 따라서 객체가 소멸될 때 까지 그 값이 유지

멤버메서드
-인스턴스가 보유한 메서드를 말한다.
-메서드란 즉 함수를 말하는데, 함수 중 특히 객체에 소속되는 함수를 가리켜 해당
객체의 동작을 표현한다고 method라 한다.

내장객체(built-in object)?
-개발자에 의해서가 아니라, 프로그래밍 언어 자체에서 유용한 기능을 위해 이미
 지원되는 객체들을 의미
 js에서는 그 종류가 Array, Date, String, Math, JSON 등이 있다

JSON 이란?
-단지 js의 문법을 차용한데이터 형식, 데이터 표기법일 뿐이다. 즉 그냥 문자열이다.
 하지만 js에서 문자열 처리를 하지 않으면 즉 "{  }" 감싸지 않고 {  } 브레이스를 그냥 코드로
 작상하면 js에서는 JSON 내장객체로 인식된다. 르래서 점찍고 메서드등 호출

DOM 이란?
-HTML모든 태그 요소마다 1:1 대응되는 객체를 의미한다.
 따라서 js, c#, java 기타 응용프로그램에서 html을 제어할수 있도록 제공되는 API이다.
